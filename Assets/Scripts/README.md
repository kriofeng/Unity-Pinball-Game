# 弹球游戏使用说明

## 如何运行游戏

1. 在Unity编辑器中打开场景（SampleScene.unity）
2. 在场景中创建一个空的GameObject，命名为"GameManager"
3. 将`PinballGameSetup`脚本添加到这个GameObject上
4. 点击Play按钮运行游戏

## 游戏控制

- **左挡板**: 按 `A` 键或 `←` 左箭头键（按住键挡板会上升）
- **右挡板**: 按 `D` 键或 `→` 右箭头键（按住键挡板会上升）
- **球**: 游戏开始时球会自动向前移动，使用挡板控制球的方向

## 视角说明

- 相机已自动设置为从上方斜视弹球台（45度角）
- 相机位置：`(0, 8, -8)`
- 相机角度：`45度俯视`
- 如果视角不对，相机会在游戏开始时自动调整

## 游戏功能

### Part 1: 物理交互
- **正常区域**（左侧灰色区域）：正常摩擦，球会正常减速
- **冰面区域**（右侧淡蓝色区域）：低摩擦，球会滑行更远

### Part 2: 粒子效果
- 当球碰撞目标区域时，会触发黄色粒子效果

### Part 3: 输入处理
- 左右两个挡板，通过键盘控制
- 挡板会向上推动球体

### Part 4: 计分系统
- 碰撞绿色目标区域获得10分
- 分数显示在屏幕左上角

## 脚本说明

- `PinballGameSetup.cs`: 主控制器，自动创建所有游戏对象和UI
- `PaddleController.cs`: 挡板控制脚本
- `ScoreManager.cs`: 计分系统
- `ParticleTrigger.cs`: 粒子效果触发器
- `PhysicsZone.cs`: 物理区域脚本（处理不同区域的物理效果）
- `UIManager.cs`: UI管理器（自动创建所有UI元素）

## 注意事项

所有游戏对象和UI都是通过脚本自动生成的，不需要手动在Unity编辑器中创建。
